# Expense Tracker

A simple command-line Expense Tracker built with Python. This project helps users manage their daily expenses by allowing them to add, view, delete, and calculate monthly expenses. All expense records are stored in a CSV file, ensuring data persists between sessions.

## Features

* Add new expenses
* View all expenses
* Delete expenses
* Update existing expenses
* Calculate monthly expense totals
* Save expenses to CSV
* Load expenses automatically from CSV on startup
* Handle missing files and invalid menu inputs
* Persistent storage using CSV files


## Technologies Used

* Python
* CSV Module
* File Handling
* Exception Handling

## Learning Outcomes

This project demonstrates the use of:

* Functions
* Lists and Dictionaries
* CSV Reading and Writing
* Exception Handling
* CRUD Operations
* Data Persistence
